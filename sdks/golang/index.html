Getting Started with the Go SDK

Rackspace's official OpenStack SDK for Go is Gophercloud, an Apache-licensed open-source community project for using multiple cloud providers from a standardized API.

Currently, Gophercloud has support for multiple Rackspace services:

  • Cloud Block Storage
  • Cloud Files
  • Cloud Servers

Support for additional services is on-going.

Requirements

  1. The Go Programming Language
  2. Git

Installation

Gophercloud is installed by executing the following command (after you have installed Go): {% highlight go %} go get github.com/rackspace/gophercloud {% endhighlight %}

Using Gophercloud

Once an Identity has been created, you specify the region when creating the Service provider. If no region is specified, any method call will fail.

In this example, we create an Identity provider that can be used to create any Service provider:

{% highlight go %} ao := gophercloud.AuthOptions{ Username: "{username}", APIKey: "{apiKey}", } provider, err := rackspace.AuthenticatedClient(ao) client, err := rackspace.NewComputeV2(provider, gophercloud.EndpointOpts{ Region: "{region}", }) {% endhighlight %}
